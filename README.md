# Busqueda tipo terminal linux 

Una **nueva pestaña minimalista estilo terminal Linux**, hecha solo con **HTML, CSS y JavaScript**.
La idea es reemplazar la típica nueva pestaña del navegador por algo **rápido, limpio y funcional**, sin trackers ni dependencias pesadas

---

## Características

* Búsqueda directa en Google
* **Autocompletado real de Google** (sugerencias oficiales)
* Navegación con teclado (`↑ ↓ Enter Esc`)
* Clima básico usando **API pública y gratuita**
* Diseño minimal, oscuro y discreto
* Sin frameworks
* Sin cuentas
* Sin permisos invasivos

---

## Tecnologías usadas

* HTML
* CSS
* JavaScript
* Google Suggest API 
* Open-Meteo API 

---

## Clima

El clima usa una **ciudad genérica (New York, USA)** por defecto 

Puedes cambiar la ciudad modificando las coordenadas en `script.js`.

---

## Cómo usarlo

### Opción 1 — Página local

1. Clona o descarga el repositorio
2. Abre `index.html` en tu navegador
3. Configúralo como **página de inicio / nueva pestaña**

### Opción 2 — Extensión de Firefox

1. Añade un `manifest.json`
2. Cárgalo desde `about:debugging`
3. Reemplaza la nueva pestaña por defecto

---

## Controles

* `Enter` → buscar en Google
* `↑ / ↓` → navegar sugerencias
* `Esc` → cerrar sugerencias

---

## Objetivo del proyecto

Este proyecto es un **experimento personal** para:

* aprender cómo funciona la nueva pestaña del navegador
* crear una interfaz más cómoda y estética
* evitar extensiones dudosas
* mantener todo simple y transparente
